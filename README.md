# NOMT — investor teaser

Live site for **NOMT**, a 15-floor mixed-use tower in central Ulaanbaatar developed by **JTSG LLC**.

Static, trilingual (Монгол / 中文 / English) single page:

- Location-first layout with a live OpenStreetMap (Leaflet) map of the site in Sükhbaatar district.
- Six income streams — for-sale residences, serviced apartments, hotel, office, retail and parking.
- Market context with sourced figures; project financials are gated ("on request").

## Files

- `index.html` — the page (markup; Tailwind via CDN with inline config)
- `styles.css` — theme and component styles
- `app.js` — trilingual content, floor explorer, contact form, and the map

No build step. Open `index.html`, or serve the folder over HTTP (the live map needs `http(s)://`, not `file://`).
